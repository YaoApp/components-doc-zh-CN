# DatePicker 日期选择框

输入或选择日期的控件。

## 使用场景

- 当用户需要输入一个日期，可以点击标准输入框，弹出日期面板进行选择。

- 展示或编辑营销等活动的截止日期

## 示例

```json
{
	"DatePicker": {
		"label": "DatePicker",
		"edit": {
			"type": "datePicker",
			"props": {
				"value": ":expired_at"
			}
		}
	}
}
```

## Props

```ts
type IProps = DatePickerProps & {
	value: string
	format?: string
}
```

## API

| 属性   | 说明   | 类型   | 默认值 | 必填 | 版本 |
| ------ | ------ | ------ | ------ | ---- | ---- |
| value  | 绑定值 | string | -      | 是   |      |
| format | 格式   | string | -      | 否   |      |

其他属性见 [Antd DatePicker](https://ant-design.gitee.io/components/datepicker-cn/)
