# CheckboxGroup 多选框

多选框组。

## 使用场景

- 在一组可选项中进行多项选择时。

- 代替 Select 组件，直接展示选项。

## 示例

```json
{
	"CheckboxGroup": {
		"label": "CheckboxGroup",
		"edit": {
			"type": "checkboxGroup",
			"props": {
				"value": ":checkboxGroup",
				"options": [
					{
						"label": "Apple",
						"value": 1
					},
					{
						"label": "Pear",
						"value": 2
					},
					{
						"label": "Orange",
						"value": 3
					}
				]
			}
		}
	}
}
```

## Props

```ts
interface Option {
	label: string
	value: string
	disabled?: boolean
}

interface IProps extends TreeProps {
	options: array<Option>
	remote: {
		api: string
		query: {
			select: Array<string>
		}
	}
}
```

## API

| 属性    | 说明         | 类型              | 默认值 | 必填 | 版本 |
| ------- | ------------ | ----------------- | ------ | ---- | ---- |
| value   | 绑定值       | string            | -      | 是   |      |
| options | 选项列表     | IProps['options'] | -      | 否   |      |
| remote  | 远程选项列表 | [Remote](#Remote) | -      | 否   |      |

### Remote

| 属性  | 说明 | 类型                      | 必填 |
| ----- | ---- | ------------------------- | ---- |
| api   | 接口 | string                    | 是   |
| query | 接口 | IProps['remote']['query'] | 否   |

其他属性见 [Antd CheckboxGroup](https://ant-design.gitee.io/components/checkbox-cn/#Checkbox-Group)
