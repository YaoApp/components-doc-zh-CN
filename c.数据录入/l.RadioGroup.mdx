# RadioGroup 多选框

多选框组。

## 使用场景

- 在一组互斥的可选项中进行单项选择时。

- 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

## 示例

```json
{
	"RadioGroup": {
		"label": "RadioGroup",
		"edit": {
			"type": "radioGroup",
			"props": {
				"value": ":radioGroup",
				"options": [
					{
						"label": "Apple",
						"value": 1
					},
					{
						"label": "Pear",
						"value": 2
					},
					{
						"label": "Orange",
						"value": 3
					}
				]
			}
		}
	}
}
```

## Props

```ts
interface Option {
	label: string
	value: string
	disabled?: boolean
}

interface IProps extends TreeProps {
	options: array<Option>
	remote: {
		api: string
		query: {
			select: Array<string>
		}
	}
}
```

## API

| 属性    | 说明         | 类型              | 默认值 | 必填 | 版本 |
| ------- | ------------ | ----------------- | ------ | ---- | ---- |
| value   | 绑定值       | string            | -      | 是   |      |
| options | 选项列表     | IProps['options'] | -      | 否   |      |
| remote  | 远程选项列表 | [Remote](#Remote) | -      | 否   |      |

### Remote

| 属性  | 说明 | 类型                      | 必填 |
| ----- | ---- | ------------------------- | ---- |
| api   | 接口 | string                    | 是   |
| query | 接口 | IProps['remote']['query'] | 否   |

其他属性见 [Antd RadioGroup](https://ant-design.gitee.io/components/radio-cn/#RadioGroup)
