# Schedule 日程表

日程表，通常用来录入或呈现一些与日程相关的信息。

## 使用场景

- 每日食用菜谱。

- 日常活动列表。

## 示例

<Notice type='warning'>注意：日程表需要配合特定的setting来使用。</Notice>

```json
{
	"Schedule": {
		"label": "Schedule",
		"edit": {
			"type": "schedule",
			"props": {
				"value": ":schedule",
				"api": "/api/xiang/setting/schedule",
				"id": ":id"
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps {
	value: string
	api: string
	id: string
}
```

## API

| 属性  | 说明             | 类型   | 默认值 | 必填 | 版本 |
| ----- | ---------------- | ------ | ------ | ---- | ---- |
| value | 绑定值           | string | -      | 是   |      |
| api   | setting 接口     | string | -      | 是   |      |
| id    | 关联的 form 变量 | string | -      | 是   |      |
