# Tree 树形控件

多层次的结构列表。

## 使用场景

文件夹、组织架构、生物分类、国家地区等等，世间万物的大多数结构都是树形结构。使用 树控件 可以完整展现其中的层级关系，并具有展开收起选择等交互功能。

- 展示和编辑若干级别的商品分类。

- 国家、省市区的多项选择。

## 示例

```json
{
	"Tree": {
		"label": "Tree",
		"edit": {
			"type": "tree",
			"props": {
				"value": ":tree",
				"checkable": true,
				"remote": {
					"api": "/api/tree/search",
					"query": {
						"select": ["id", "name"]
					}
				}
			}
		}
	}
}
```

## Props

```ts
interface Option {
	key: string
	title: string
	children: Array<Option>
	disabled?: boolean
	selectable?: boolean
}

interface IProps extends TreeProps {
	options: array<Option>
	remote: {
		api: string
		query: {
			select: Array<string>
		}
	}
}
```

## API

| 属性    | 说明         | 类型              | 默认值 | 必填 | 版本 |
| ------- | ------------ | ----------------- | ------ | ---- | ---- |
| value   | 绑定值       | string            | -      | 是   |      |
| options | 选项列表     | IProps['options'] | -      | 否   |      |
| remote  | 远程选项列表 | [Remote](#Remote) | -      | 否   |      |

### Remote

| 属性  | 说明 | 类型                      | 必填 |
| ----- | ---- | ------------------------- | ---- |
| api   | 接口 | string                    | 是   |
| query | 接口 | IProps['remote']['query'] | 否   |

其他属性见 [Antd Tree](https://ant-design.gitee.io/components/tree-cn/)
