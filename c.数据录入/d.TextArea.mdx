# TextArea 多行文本编辑控件

TextArea，支持输入一段相当长的、不限格式的文本。

## 使用场景

- 网站评论。

- 一些详细的描述，如用户地址等。

- 反馈表单中的一段意见。

## 示例

```json
{
	"送货地址": {
		"label": "送货地址",
		"edit": {
			"type": "textArea",
			"props": {
				"value": ":address",
				"rows": 3
			}
		}
	}
}
```

## Props

```ts
interface IProps extends TextAreaProps {
	value: string
	label?: string
}
```

## API

| 属性  | 说明     | 类型   | 默认值 | 必填 | 版本 |
| ----- | -------- | ------ | ------ | ---- | ---- |
| value | 绑定值   | string | -      | 是   |      |
| label | 字段标题 | string | -      | 否   |      |

其他属性见 [Antd TextArea](https://ant-design.gitee.io/components/input-cn/#Input.TextArea)
