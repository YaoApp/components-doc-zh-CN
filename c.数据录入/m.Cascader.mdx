# Cascader 级联选择

级联选择框。

## 使用场景

- 需要从一组相关联的数据集合进行选择，例如省市区，公司层级，事物分类等。

- 从一个较大的数据集合中进行选择时，用多级分类进行分隔，方便选择。

- 比起 Select 组件，可以在同一个浮层中完成选择，有较好的体验。

## 示例

```json
{
	"Cascader": {
		"label": "Cascader",
		"edit": {
			"type": "cascader",
			"props": {
				"value": ":cascader",
				"showSearch": true,
				"placeholder": "请选择所属厂商",
				"remote": {
					"api": "/api/cascader/search",
					"query": {
						"select": ["id", "name"]
					}
				}
			}
		}
	}
}
```

## Props

```ts
interface Option {
	value: string | number
	label?: React.ReactNode
	disabled?: boolean
	children?: Array<Option>
}

interface IProps extends TreeProps {
	options: array<Option>
	remote: {
		api: string
		query: {
			select: Array<string>
		}
	}
}
```

## API

| 属性    | 说明         | 类型              | 默认值 | 必填 | 版本 |
| ------- | ------------ | ----------------- | ------ | ---- | ---- |
| value   | 绑定值       | string            | -      | 是   |      |
| options | 选项列表     | IProps['options'] | -      | 否   |      |
| remote  | 远程选项列表 | [Remote](#Remote) | -      | 否   |      |

### Remote

| 属性  | 说明 | 类型                      | 必填 |
| ----- | ---- | ------------------------- | ---- |
| api   | 接口 | string                    | 是   |
| query | 接口 | IProps['remote']['query'] | 否   |

其他属性见 [Antd Cascader](https://ant-design.gitee.io/components/cascader-cn/)
