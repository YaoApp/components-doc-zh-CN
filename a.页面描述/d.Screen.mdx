# Screen 大屏

大屏，常用于投影到大屏设备，全屏显示相关图表。

## 使用场景

- 展示关键性指标

- 实时数据监测

## Props

<Notice type='success'>
	大屏配置项同Chart，区别是，大屏没有筛选栏，且大屏多出顶部指标栏，以及大屏的布局为grid布局，且纵向排布，需同时设定
	width 和 height，width 为24栅格，height 为12栅格。
</Notice>

```ts
interface IProps {
	page: {
		// 顶部指标栏数据
		option: {
			left_index: ['云计算行业总产值', '云计算相关企业', '持牌企业', '中国云计算份额']
			right_index: ['产业月增长率', '产业季度增长率', '产业年增长率', '市场需求增长率']
		}
	}
}
```
