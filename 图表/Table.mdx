# Table 表格

在图表页展示一些结构化数据。

## 示例

```json
{
	"name": "云厂商概览",
	"width": 12,
	"type": "table",
	"props": {
		"height": 240,
		"columns": [
			{
				"title": "名称",
				"dataIndex": "name"
			},
			{
				"title": "使用率",
				"dataIndex": "usage"
			},
			{
				"title": "积分",
				"dataIndex": "score"
			},
			{
				"title": "市场占有率",
				"dataIndex": "rate"
			}
		]
	}
}
```

## Props

```ts
interface IProps {
	height: number
	columns: Array<ColumnsType<any>>
	autoplay?: boolean
	speed?: number
}
```

## API

| 属性     | 说明     | 类型   | 默认值 | 必填 | 版本 |
| -------- | -------- | ------ | ------ | ---- | ---- |
| height   | 高度     | string | -      | 是   |      |
| columns  | 数据列   | string | -      | 是   |      |
| autoplay | 自动滚动 | string | -      | 否   |      |
| speed    | 滚动速度 | string | -      | 否   |      |

Column 属性见 [Antd Table Column](https://ant-design.gitee.io/components/table-cn/#Column)
