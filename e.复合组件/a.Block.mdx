# Block 块

块，用于展示图像、标题和描述。

## 示例

<Notice type='warning'>注意：components属性中的value将会被解析成columns预先定义的组件。</Notice>

```json
{
	"服务信息": {
		"label": "服务信息",
		"view": {
			"type": "block",
			"components": {
				"avatar": "营销图片",
				"name": "服务名称",
				"desc": "服务类型"
			}
		}
	}
}
```

## Props

```ts
interface IProps {
	avatar: string
	name: string
	desc: string
}
```

## API

| 属性   | 说明 | 类型   | 默认值 | 必填 | 版本 |
| ------ | ---- | ------ | ------ | ---- | ---- |
| avatar | 图片 | string | -      | 是   |      |
| name   | 名称 | string | -      | 是   |      |
| desc   | 简介 | string | -      | 是   |      |
