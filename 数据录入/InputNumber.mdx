# InputNumber 数字输入框

通过鼠标或键盘，输入范围内的数值。

## 使用场景

- 当需要获取标准数值时。

- 提供格式化的数字输入，以此来约束输入内容。

## 示例

```json
{
	"InputNumber": {
		"label": "InputNumber",
		"edit": {
			"type": "inputNumber",
			"props": {
				"value": ":amount",
				"placeholder": "请输入额度"
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputNumberProps {
	value: string
}
```

## API

| 属性  | 说明     | 类型   | 默认值 | 必填 | 版本 |
| ----- | -------- | ------ | ------ | ---- | ---- |
| value | 绑定值   | string | -      | 是   |      |

其他属性见 [Antd InputNumber](https://ant-design.gitee.io/components/input-number-cn/)
