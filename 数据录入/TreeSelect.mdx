# TreeSelect 树形选择框

树型选择控件。

## 使用场景

类似 Select 的选择控件，可选择的数据结构是一个树形结构时，可以使用 TreeSelect，例如公司层级、学科系统、分类目录等等。

- 选择公司部门。

- 选择指定类目。

## 示例

```json
{
	"TreeSelect": {
		"label": "TreeSelect",
		"edit": {
			"type": "treeSelect",
			"props": {
				"value": ":treeSelect",
				"allowClear": true,
				"multiple": true,
				"remote": {
					"api": "/api/tree/search",
					"query": {
						"select": ["id", "name"]
					}
				}
			}
		}
	}
}
```

## Props

```ts
interface Option {
	key: string
	title: string
	children: Array<Option>
	disabled?: boolean
	disableCheckbox?: boolean
	selectable?: boolean
	checkable?: boolean
}

interface IProps extends TreeProps {
	options: array<Option>
	remote: {
		api: string
		query: {
			select: Array<string>
		}
	}
}
```

## API

| 属性    | 说明         | 类型              | 默认值 | 必填 | 版本 |
| ------- | ------------ | ----------------- | ------ | ---- | ---- |
| value   | 绑定值       | string            | -      | 是   |      |
| options | 选项列表     | IProps['options'] | -      | 否   |      |
| remote  | 远程选项列表 | [Remote](#Remote) | -      | 否   |      |

### Remote

| 属性  | 说明 | 类型                      | 必填 |
| ----- | ---- | ------------------------- | ---- |
| api   | 接口 | string                    | 是   |
| query | 接口 | IProps['remote']['query'] | 否   |

其他属性见 [Antd TreeSelect](https://ant-design.gitee.io/components/tree-select-cn/)
