# Input 输入框

通过鼠标或键盘输入内容，是最基础的表单域的包装。

## 使用场景

- 需要用户输入表单域内容时。

- 提供组合型输入框，带搜索的输入框，还可以进行大小选择。

## 示例

```json
{
	"Input": {
		"label": "Input",
		"edit": {
			"type": "input",
			"props": {
				"value": ":group",
				"placeholder": "请输入分组"
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps {
	value: string
}
```

## API

| 属性  | 说明   | 类型   | 默认值 | 必填 | 版本 |
| ----- | ------ | ------ | ------ | ---- | ---- |
| value | 绑定值 | string | -      | 是   |      |
