# ConditionList 条件列表

条件列表，支持在 Form 中对指定类型数据列表做 CURD 操作并支持联动。

## 示例

<Notice type='warning'>注意：条件列表需要配合特定的setting来使用。</Notice>

```json
{
	"ConditionList": {
		"label": "ConditionList",
		"edit": {
			"type": "conditionList",
			"props": {
				"setting": "survey",
				"value": ":survey",
				"query": {
					"sort": "$index"
				}
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps {
	value: string
	setting: string
	query?: { [key: string]: string }
}
```

## API

| 属性    | 说明                                         | 类型            | 默认值 | 必填 | 版本 |
| ------- | -------------------------------------------- | --------------- | ------ | ---- | ---- |
| value   | 绑定值                                       | string          | -      | 是   |      |
| setting | setting 名称                                 | string          | -      | 是   |      |
| query   | 提交参数，将会在提交数据时原样整合到 data 中 | IProps['query'] | -      | 否   |      |
