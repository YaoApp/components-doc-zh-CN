# Item 表单项

Item 是所有数据录入组件的父组件，Item 通过事件劫持的方式来对表单组件进行统一的处理。

## 使用场景

<Notice type='warning'>注意：该组件无法单独使用，数据录入类型的组件会自带 Item。</Notice>

## 示例

```json
{
	"Item": {
		"label": "Item",
		"edit": {
			"type": "input",
			"props": {
				"value": ":group",
				"rules": [
					{
						"required": true
					}
				]
			}
		}
	}
}
```

## Props

```ts
interface IProps extends FormItemProps {
	hide_label?: boolean
}
```

## API

| 属性       | 说明           | 类型    | 默认值 | 必填 | 版本 |
| ---------- | -------------- | ------- | ------ | ---- | ---- |
| hide_label | 是否显示 label | boolean | -      | 否   |      |

其他属性见 [Antd Form.Item](https://ant-design.gitee.io/components/form-cn/#Form.Item)
